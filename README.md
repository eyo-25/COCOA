## 🔎 Service Intro

Cocoa는 코인 정보 플랫폼으로써 실시간 코인 정보를 제공하며 코인별 시세 변동 차트와 코인과 관련된 뉴스 피드를 제공하여 소비자들로 하여금 코코아처럼 달달한 코인 정보를 받아 볼 수 있도록 합니다.
